# xstart2

#### 介绍
一款使用wpf开发的，仿vstart(音速启动)的应用软件

#### 软件架构
使用.net framework 4.5的wpf架构

引用插件\
1.CliWrap --执行插件\
2.ini-parser --ini文件解析\
3.NAudio     -- 声音播放\
4.Newtonsoft.Json    --json串处理\
5.PropertyChanged.Fody    --属性通知\
6.WpfAnimatedGif           --gif动画\
7.MSTSCLib       --远程插件

#### 功能点
快捷应用自启动\
软体主题变更\
文件，目录，系统功能快捷启动\
日历，天气插件\
远程桌面统一管理
